Редактор презентаций, работающий прямо в браузере — без установки, без сервера, без интернета.

---

## Быстрый старт

Откройте `index.html` в браузере. Всё готово к работе.

---

## Интерфейс

```
┌─────────────┬──────────────────────────────┬───────────────┐
│  Слайды     │         Канвас               │  Свойства     │
│  (список)   │      1200 × 675 px           │  элемента     │
│             │                              │               │
│             │                              │  Анимации     │
└─────────────┴──────────────────────────────┴───────────────┘
```

- **Левая панель** — список слайдов, добавление/удаление, порядок (drag)
- **Канвас** — рабочая область, масштабируется колесом мыши
- **Правая панель** — свойства выбранного элемента + вкладка «Анимации»
- **Верхняя панель** — инструменты вставки, цветовая схема, макет, предпросмотр

## Элементы

### Надпись
- Двойной клик на канвасе — создать надпись
- Двойной клик на надписи — войти в режим редактирования
- Rich text: выделите текст → выберите цвет, жирный, курсив, подчёркивание
- Цвет берётся из палитры цветовой схемы (позиция запоминается)

### Изображение
- Вставить через панель или перетащить файл
- **Кроп:** выберите изображение → кнопка «Обрезать» → тяните ручки → «Применить»
- Настройки: радиус углов, обводка, тень, прозрачность

### Markdown
- Вставить через панель → откроется редактор
- **Двойной клик** по блоку — снова открыть редактор
- В панели свойств: размер шрифта, цвет текста (из палитры схемы)
- Цвет автоматически меняется при смене цветовой схемы

### Код
- Вставить через панель → выбрать язык → вставить код
- **Двойной клик** по блоку — редактировать
- Тема кода синхронизируется с цветовой схемой (тёмная/светлая)

### Фигура
- Выбрать форму в панели → вставить
- Заливка и обводка берутся из палитры схемы

### Таблица, Иконка, Аплет
- Вставить через соответствующие кнопки в верхней панели

## Цветовые схемы

Кнопка **«Цвет.схема»** (или иконка палитры) → окно выбора.

- 38 схем: 22 тёмных, 16 светлых
- **Клик по схеме** — применяет сразу, окно остаётся открытым (можно перебирать)
- **«Применить ко всем»** — применяет и закрывает окно
- При смене схемы все цвета, выбранные из палитры, автоматически адаптируются

### Палитра цветов

Открывается при клике на цветовой свотч в панели свойств.

- 8 колонок × 5 строк
- Колонки 1–7: цвета схемы с тинтами от насыщенного к светлому
- **Последняя колонка:** тёмная тема — белый→чёрный, светлая — чёрный→белый
- Кастомный цвет — через `input[type=color]` внизу (не привязывается к схеме)

## Макеты (декор)

Кнопка **«Тема»** → декоративный фон для всех слайдов.

- 14 стилей: Prism, Aurora, Circuit, Halo, Space и др.
- **«Без декора»** — карточка с крестиком, убирает декор
- Цвета декора берутся из текущей цветовой схемы

---

## Анимации

Вкладка **«Анимации»** в правой панели (при выбранном элементе).

- **Вход / Выход / Акцент** — категории эффектов
- **Триггер:** «По клику», «С предыдущим», «После предыдущего»
- **Задержка и длительность** — настраиваются для каждой анимации
- В режиме показа клик на слайд запускает следующий шаг анимации

---

## Показ (F5)

- `F5` или кнопка ▶ — запустить показ
- `← →` или клик — следующий/предыдущий слайд
- `Esc` — выйти
- **clickNav = выкл.** на слайде — клик запускает только анимации, не меняет слайд

---

## Экспорт

- **HTML** — полностью автономный файл, работает без редактора
- Экспортированный файл включает все анимации, переходы и кроп изображений

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `Ctrl+Z` | Отменить |
| `Ctrl+Y` | Повторить |
| `Ctrl+C` | Копировать |
| `Ctrl+V` | Вставить |
| `Ctrl+D` | Дублировать |
| `Ctrl+A` | Выделить всё |
| `Delete` | Удалить |
| `F5` | Показ |
| `Esc` | Выйти из показа |
| `↑↓←→` | Сдвинуть на 1px |
| `Shift+↑↓←→` | Сдвинуть на 10px |

## Настройки

**Шестерёнка** в углу → настройки:
- Тёмная / Светлая тема интерфейса
- Язык: Русский / English
- Нумерация страниц

---

## Технические детали

- Все данные хранятся в `localStorage` браузера
- Размер канваса: 1200 × 675 px (16:9)
- Нет зависимостей от сети, CDN или сервера
- Совместим с современными браузерами (Chrome, Firefox, Edge, Safari)
