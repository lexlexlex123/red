# Слайды — Документация проекта

## Текущая версия: v4.7
**Автор:** Некрасов Александр

> Презентационный редактор в одном HTML-файле (или разбитый на модули).  
> Работает без сервера, без сборки — просто открой `index.html` в браузере.  
> Все библиотеки (JSZip, QRCode) размещены локально — работает полностью офлайн.

---

## Что нового в v4.7

- **Мультиязычный интерфейс** — переключение между русским и английским прямо в настройках. Язык сохраняется между сессиями
- **Переименование** — программа теперь называется **«Слайды»**
- **Версия и автор** в настройках — отображаются версия программы и имя автора
- **32 темы оформления** — 18 тёмных и 14 светлых, разделены на секции с превью цветов
- **Цветовые метки** на карточках тем — два квадратика: цвет заголовков и цвет акцентов/фигур
- **Нумерация страниц** — аплет с 6 стилями, позиционированием по кнопкам или перетаскиванием, цвет из темы или вручную, размер и прозрачность; положение запоминается и применяется ко всем слайдам
- **Офлайн-режим** — JSZip и QRCode подключены локально из `libs/`, Google Fonts заменены системными шрифтами
- **Исправления сохранения** — переписана логика `save/saveState/commitAll`: DOM→данные и данные→localStorage разделены, перед F5 всегда делается полный флаш
- **Исправлен цвет текста** — смена цвета, жирного, курсива через выделение или целиком теперь корректно сохраняется и не сбрасывается после показа
- **Исправлено вертикальное выравнивание текста** — flex-контейнер больше не разбивает символы в столбик; используется внутренний wrapper-div
- **Возврат слайда после показа** — F5, Esc и кнопка ✕ возвращают именно на тот слайд, где остановился показ

## Возможности

### Элементы
- **Текст** — шрифт, цвет, размер, жирный/курсив/подчёркивание, выравнивание по горизонтали и вертикали, интерлиньяж, межбуквенный интервал, роль (заголовок / текст)
- **Изображения** — загрузка, обрезка, скругление, тень, прозрачность, позиционирование
- **Фигуры** — 16 видов с заливкой, обводкой, скруглением, тенями
- **Иконки** — 500+ SVG-иконок с поиском, выбором стиля и цвета
- **SVG** — вставка произвольного SVG-кода
- **Code blocks** — подсветка синтаксиса (JS, Python, HTML, CSS, SQL, Bash)
- **Markdown** — рендеринг прямо на слайде
- **Таблицы** — ячейки с форматированием, изменение размера колонок/строк, импорт из PPTX
- **Аплеты** — встроенные интерактивные виджеты (заметки, QR-код, диаграмма и др.)
- **Нумерация страниц** — 6 стилей, позиционирование кнопками или drag, цвет из темы или вручную

### Дизайн
- **32 темы** (18 тёмных + 14 светлых) с цветовыми метками акцентов
- **Фоны** — градиенты, сплошные цвета, произвольные
- **Декоратор макетов** — готовые компоновки слайдов

### Анимации и переходы
- 11 анимаций входа/выхода элементов
- 8 эффектов перехода между слайдами (включая Morph)
- Настройка тайминга, задержки, автоперехода
- Режим триггера — анимации по клику на элемент

### Показ
- Полноэкранный режим показа (F5), возврат на слайд остановки
- Поддержка ссылок-навигации между слайдами
- Экспорт в самодостаточный HTML-файл
- Импорт из PPTX (слайды, текст, изображения, таблицы)

### Интерфейс
- **Язык** — 🇷🇺 Русский / 🇬🇧 English (переключение в ⚙ Параметры)
- Тёмная и светлая тема интерфейса
- Привязка к сетке 10px, направляющие
- Undo/Redo до 40 шагов (Ctrl+Z / Ctrl+Y), включая нумерацию
- Множественное выделение (Shift+клик, Ctrl+A), групповое перемещение
- Выравнивание и распределение элементов
- Автосохранение сессии в localStorage

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `Ctrl+Z` | Отменить |
| `Ctrl+Y` / `Ctrl+Shift+Z` | Повторить |
| `Ctrl+C` | Копировать элемент |
| `Ctrl+V` | Вставить элемент |
| `Ctrl+D` | Дублировать элемент |
| `Ctrl+A` | Выделить всё на слайде |
| `Delete` | Удалить элемент |
| `F5` | Начать показ / выйти (возврат на текущий слайд) |
| `Esc` | Выйти из режима показа (возврат на текущий слайд) |
| `←` `→` | Предыдущий/следующий слайд (в режиме показа) |
| `↑` `↓` `←` `→` | Сдвинуть элемент на 1px (Shift — на 10px) |

## Архитектура сохранения (js/27-persist.js)

```
save()       — DOM → slides[cur].els  (только память, без I/O)
saveState()  — slides[] → localStorage (только I/O, без DOM-чтения)
commitAll()  — save() + drawThumbs() + saveState()
```

**Правило:** `commitAll()` вызывается после каждого действия пользователя.  
`saveState()` без `save()` — только там где DOM не при чём (переходы, undo, импорт).  
Перед F5 всегда вызывается `commitAll()` — гарантирует что всё сохранено до fullscreen.

## Нумерация страниц (js/33-pagenum.js)

```js
pnSettings = {
  enabled, style, position, customXY,   // позиция: preset или {x,y} после drag
  color, textColor, fontSize, opacity,
  showTotal, customColor
}

pnApplyAll()   // применить/обновить на всех слайдах
pnSetPos(pos)  // preset-кнопка (сбрасывает customXY)
pnOnDragEnd(x,y) // после ручного перетаскивания
pnLock()       // заморозить на время показа (F5)
pnUnlock()     // разморозить + pnSyncUI()
```

## Система локализации (js/00-i18n.js)

```js
t('btnNewSlide')   // → 'New Slide' или 'Новый слайд'
setLang('en')      // переключить на английский
getLang()          // → 'ru' или 'en'
```

В HTML используются атрибуты `data-i18n`, `data-i18n-placeholder`, `data-i18n-title`.
