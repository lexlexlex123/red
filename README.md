## Возможности

### Элементы слайда
| Тип | Описание |
|-----|----------|
| **Текст** | Форматированный редактируемый текст, heading / body / caption |
| **Изображение** | PNG / JPG / SVG, crop, rx, opacity |
| **Фигура** | 30+ фигур включая callout с хвостом |
| **Таблица** | Редактируемые ячейки, стили границ |
| **Code-блок** | Синтаксическая подсветка (JS, Python, HTML, CSS и др.) |
| **Формула** | LaTeX через MathJax |
| **Иконка** | 1000+ иконок, stroke / fill / duotone |
| **Граф** | Интерактивный график (Chart.js) |
| **Веб-элемент** | iframe с URL или HTML-кодом (`htmlframe`) |
| **Соединитель** | Bezier-линии между элементами с маркерами |

### Редактор
- Undo / Redo (Ctrl+Z / Ctrl+Y)
- Выравнивание и распределение элементов
- Вращение, масштаб, opacity
- Множественное выделение (rubber-band)
- Горячие клавиши: `T` текст, `I` изображение, `S` фигура, `Del` удалить, `F` fullscreen
- Темы оформления (8 встроенных)
- Анимации входа / выхода элементов
- Переходы между слайдами

### Экспорт
- **HTML** — автономный файл, открывается в любом браузере
  - Все элементы включая веб-фреймы, формулы, графики
  - Поддержка анимаций, переходов, автопрокрутки
  - Fullscreen по `F`, навигация стрелками / кликом
- **PPTX** — импорт PowerPoint файлов

## Веб-элемент (htmlframe) — новое в v5.4

Добавляет в слайд живой iframe с браузерным chrome.

**Создание:** кнопка `<html>` в тулбаре → вставить URL или HTML-код.

**В редакторе:** прозрачный overlay блокирует случайные клики — элемент можно перемещать и ресайзить как обычно.

**В показе / экспорте:** iframe полностью интерактивен. Зелёная кнопка разворачивает на весь слайд.

**Связанный code-блок:** кнопка «📄 Показать код» в панели свойств создаёт рядом code-блок с HTML-кодом; при редактировании htmlframe code-блок обновляется автоматически.

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `T` | Добавить текст |

| `Space` | Следующий слайд (в показе) |

## Технические детали

- **Без зависимостей** в рантайме — чистый JS/HTML/CSS
- **localStorage** — автосохранение состояния (`sf_v4`)
- **Модульная структура** — каждый файл независим, загружаются через `<script src>`
- **Экспорт** — `buildExportHTML()` в `26-export.js` генерирует единый самодостаточный HTML
- Данные слайдов передаются через `<script type="application/json">` (без escaping-проблем)
