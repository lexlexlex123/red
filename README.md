## Быстрый старт

Распакуйте архив и откройте `index.html` в браузере. Интернет не требуется.

```
slides_project/
├── index.html
├── css/styles.css
├── images/
│   ├── image-index.js      — реестр изображений галереи
│   ├── build-index.js      — генератор реестра
│   └── nature/ animals/ birds/ business/ ...
└── js/
    ├── 00-i18n.js          — локализация, APP_VERSION
    ├── 09-shapes.js        — фигуры, скругление, стили границ
    ├── 10-animations.js    — панель анимаций
    ├── 15-align.js         — выравнивание (поддержка групп)
    ├── 17-text.js          — стили границы текста, пипетка
    ├── 26-export.js        — экспорт в HTML
    ├── 44-group.js         — группировка, вращение, выравнивание
    └── ...
```

## Возможности

### Элементы
| Тип | Описание |
|-----|----------|
| **Текст** | Форматированный текст, 6 стилей границы |
| **Изображение** | PNG/JPG/SVG, crop, flip H/V, rx, opacity |
| **Фигура** | 30+ фигур, скругление углов (Bezier), 6 стилей границы |
| **Таблица** | Редактируемые ячейки |
| **Code-блок** | Синтаксическая подсветка |
| **Формула** | LaTeX (MathJax) |
| **Иконка** | 1000+ иконок |
| **Граф** | Chart.js |
| **Веб-элемент** | iframe |
| **Соединитель** | Bezier |

### Стили границы (фигуры и текст)
Сплошная · Пунктирная · Точками · Двойная · Волнистая · Зигзаг

### Группировка
- `Ctrl+G` / `Ctrl+Shift+G` — группировать / разгруппировать
- Перемещение, масштабирование, вращение группы как единого целого
- Зоны вращения в углах bbox (как для одиночных элементов)
- Выравнивание группы не рассыпает элементы

| `Ctrl+Shift+G` | Разгруппировать |
| `F` | Показ / fullscreen |
