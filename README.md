## Что нового в v5.5

### Галерея изображений
Модальное окно с категориями, поиском и превью. Открывается кнопкой «Изображение» в ленте.

**Настройка:**
1. Положите файлы в `images/{категория}/`
2. Запустите `update-gallery.bat` (или `node images/build-index.js`)
3. Обновите страницу

Категории: `nature`, `animals`, `birds`, `business`, `technology`, `backgrounds`, `textures`, `people`, `abstract`, `icons_png`

**Формат реестра:**
```js
// Минимальный:
{id:'img1', cat:'birds', file:'eagle.jpg', name:'Орёл'}

// SVG со встроенным контентом (вставляется как редактируемый вектор):
{id:'img2', cat:'abstract', file:'wave.svg', name:'Волна',
 svgContent: `<svg ...>...</svg>`}
```

### Отражение изображений
Кнопки «Отразить по горизонтали» и «Отразить по вертикали» в панели свойств изображения. Флип корректно работает в редакторе, предпросмотре и экспорте.

### Анимация «Качение» 🎷
Новая живая анимация в категории «Живая». Объект раскачивается с затуханием: 30°→−30°→20°→−20°→10°→−10° и возвращается в исходное положение.

- **Центр качения** — фиолетовая точка прямо на объекте, перетаскивается мышью
- **Snap** — притягивается к углам, серединам сторон и центру объекта (16px)
- **Поверх маркеров** — точка не перекрывается маркерами resize
- **Количество повторений** — числовое поле (1–9) + чекбокс «∞»
- **Совместимость** — работает одновременно с moveTo («Вместе с предыдущим»)

### Двойной клик по эффекту
Двойной клик по названию анимации в панели мгновенно добавляет её к выбранному объекту.

## Возможности

### Элементы
| Тип | Описание |
|-----|----------|
| **Текст** | Форматированный редактируемый текст |
| **Изображение** | PNG / JPG / SVG, crop, flip H/V, rx, opacity |
| **Фигура** | 30+ фигур, callout |
| **Таблица** | Редактируемые ячейки |
| **Code-блок** | Синтаксическая подсветка |
| **Формула** | LaTeX (MathJax) |
| **Иконка** | 1000+ иконок |
| **Граф** | Chart.js |
| **Веб-элемент** | iframe |
| **Соединитель** | Bezier |

### Анимации
| Категория | Эффекты |
|-----------|---------|
| Вход | fadeIn, slideUp/Down/Left/Right, zoomIn, bounceIn, spinIn |
| Выход | fadeOut, slideOut, zoomOut |
| Акцент | pulse, shake, flash, rotate |
| Движение | moveTo, orbitTo |
| Живая | Танец, **Качение** 🆕, Смена текста |

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `T` | Добавить текст |
| `I` | Открыть галерею изображений |
| `S` | Открыть галерею фигур |
| `Del` | Удалить элемент |
| `Ctrl+Z / Y` | Отменить / Повторить |
| `Ctrl+D` | Дублировать |
| `Ctrl+A` | Выбрать все |
| `F` | Показ / fullscreen |
| `←→` | Навигация по слайдам |
