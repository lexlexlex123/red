# Слайды — v5.8

Веб-редактор презентаций. Работает полностью в браузере, без серверной части.  
**Автор:** Некрасов Александр

---

## Быстрый старт

Распакуйте архив и откройте `index.html` в браузере. Интернет не требуется.

```
slides_project/
├── index.html
├── css/styles.css
└── js/
    ├── 00-i18n.js          — локализация, APP_VERSION
    ├── 09-shapes.js        — фигуры, скругление, стили границ
    ├── 10-animations.js    — анимации (качение, танец, плавание)
    ├── 41-lego.js          — LEGO-конструктор
    ├── 44-group.js         — группировка
    ├── 45-media.js         — аудио/видео
    └── 50-voice.js         — голосовое управление
```

## Возможности

### Элементы
| Тип | Описание |
|-----|----------|
| **Текст** | Форматированный, 6 стилей границы |
| **Изображение** | PNG/JPG/SVG, crop, flip, rx, opacity |
| **Фигура** | 30+ фигур, скругление Bezier, 6 стилей границы |
| **Линия / Волна** | Без заливки, широкая область клика |
| **Таблица** | Редактируемые ячейки |
| **Формула** | LaTeX (MathJax) |
| **Иконка** | 1000+ иконок |
| **Граф** | Chart.js |
| **Соединитель** | Bezier |
| **Аудио / Видео** | Встроенные медиа с триггерами |
| **LEGO** | Кирпичики, скосы, обратные скосы |

### Живые анимации
- **Качение** — маятниковое покачивание
- **Танец** — сжатие/растяжение
- **Плавание 🌊** — органический дрейф, бесшовный loop

### Голосовое управление 🎙
200+ команд на русском. Повелительное и инфинитивное наклонение:
- Объекты: создать, удалить, дублировать, выбрать по роли/позиции
- Свойства: цвет, размер, поворот, прозрачность, шрифт
- Выравнивание, группировка, темы, переходы
- Навигация в режиме показа
- Журнал нераспознанных команд 📋

### Магнитное выделение
Клик рядом с объектом (радиус 40px) автоматически выделяет ближайший элемент.

### Группировка
`Ctrl+G` / `Ctrl+Shift+G` — группировать / разгруппировать. Масштаб, вращение, выравнивание как единого целого.
